100% Open Source

FerrisChat is completely open-source, so that you can know exactly what it does. If you have a problem with the way it is performing, or maybe you don't like the color, you can just go into the code and change it! If you feel that others may enjoy your change, then simply create a pull request.

Created with Privacy in Mind

We will NEVER sell any of your data, nor give it to anyone. Period.

Powered by Ferris

The entire program is written in Rust, a new language that specializes in speed and safety. You no longer need an insane computer to run a messenger, nor worry about the code causing problems for your computer.
